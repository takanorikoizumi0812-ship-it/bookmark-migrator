# Bookmark Migrator

Safariのお気に入りを、Chrome, Vivaldi, Braveなどのブラウザへ一括で移行できるMac用アプリです。

## 🚀 ダウンロードと使い方
1. 画面右側の **「Releases」** から、最新の `.zip` ファイルをダウンロードして解凍します。
2. **【重要】** 初回起動時は、アプリアイコンをそのままダブルクリックせず、**右クリック（またはControlキーを押しながらクリック）して「開く」を選択**してください。（開発元が未確認という警告が出ますが、もう一度「開く」を押すと起動できます）

## ⚠️ 必須設定（フルディスクアクセス）
Safariのデータにアクセスするため、Macのシステム設定が必要です。
1. Macの「システム設定」>「プライバシーとセキュリティ」>「フルディスクアクセス」を開く。
2. リストの下にある「＋」ボタンを押し、ダウンロードしたこのアプリを選択して権限をオンにしてください。
3. アプリを一度終了（Cmd + Q）して、再度起動すると移行機能が使えるようになります。
